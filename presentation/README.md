# CRESCENDO LABS - One Pager Presentation

This is the official one-slide presentation for the CRESCENDO LABS project, built with [Slidev](https://slidev.js.org/).

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Export as PDF
npm run export
```

## Presentation Content

The presentation covers:
- **Problem & Solution:** IMSS lab appointment bureaucratic challenges
- **Architecture:** OpenClaw domain with Gmail integration
- **Proposed Flow:** From doctor visit to automated reminders
- **Key Metrics:** Token efficiency and security guardrails
- **Tech Stack:** HealthTech vertical with accessibility focus

Open http://localhost:3030 after running `npm run dev`
